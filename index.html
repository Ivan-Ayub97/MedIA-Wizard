MedIA-Wizard

AI-Powered Image and Video Upscaling Tool

MedIA-Wizard is a powerful open-source desktop application designed to enhance image and video quality and resolution using advanced AI models. Built on the powerful foundation of RealScaler, this tool provides an optimized media upscaling experience with state-of-the-art models such as Real-ESRGAN, Waifu2x, and Anime4K.

  • Advanced AI models for upscaling: Utilizes Real-ESRGAN, Waifu2x, Anime4K, and other optimized models to improve image and video quality with sharp details and noise reduction.
  • Batch processing: Upscale multiple images and videos simultaneously, ideal for restoring entire collections.
  • Customizable settings: Adjust output resolution, image/video format, and select the most suitable AI model for the best results.
  • User-friendly interface: Designed for users of all levels, making media enhancement easy and fast.

üöÄ Installation

To get started with MedIA-Wizard, follow these steps:

  1. Run the installer and follow the on-screen instructions.
  2. Launch the application: Run MedIA-Wizard.exe on your Windows system.
  3. Start enhancing your media!

MedIA-Wizard uses PyInstaller and InnoSetup for simplified installation and executable creation.

üõ†Ô∏è How to Use

  1. Launch the application: Run MedIA-Wizard.exe as an administrator for optimal performance.
  2. Select your media files: Upload images, videos, or entire folders for enhancement.
  3. Configure your settings: Choose the AI model (Real-ESRGAN, Waifu2x, Anime4K, etc.) and adjust the output resolution and format (PNG, JPEG, MP4, etc.).
  4. Start the enhancement process: Click the UPSCALE button to begin.
  5. Review the results: Once the processing is complete, your enhanced media will be saved in the output folder.
